Over mij

Wie ben ik?

Ik ben Abdalaziz, een tweedejaars software developer student (MBO 4) die graag hands-on leert. Van encryptie algoritmes tot game development — ik pak uitdagingen aan door gewoon te beginnen en te leren onderweg.

Momenteel werk ik aan cybersecurity opdrachten waarbij ik encryptie, hashing, JWT authenticatie en SQL injection beveiliging toepas. Daarnaast bouw ik een 3D poly horror game in Unity.

C# is mijn hoofdtaal, maar ik heb ook ervaring met JavaScript, Java, SQL en HTML/CSS. Ik gebruik Visual Studio, WebStorm en Unity dagelijks in mijn projecten.
